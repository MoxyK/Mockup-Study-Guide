HTML

  • Head element contains information about the webpage
  • Body element represents the visible content shown to the user

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside and element.

Git

  • git status: Checks what branch we are currently on
  • git checkout: -b branch-name: Creates a new branch and switches to it
  • git reset --merge: Will reset the index and update the files in the working tree that are different between the commit and the head. However, it will keep those files which are different between the index and working tree.
  • git pull & git push: Sends push and pull signals to other branches and brings content up to date within working tree
  • Saving branches: git add -A :: git commit -m "added *** to project" :: git pull origin main :: git push origin 'name of branch to add' :: This will send notification to your Git Repo to approve the branch merge
  • git checkout main :: git pull origin main: Updates local main branch in line with remote main branch

JavaScript

  • A variable is a named container that allows us to store data in our code
  • Array: A single variable that is used to hold a group of data. Arrays are typically used to hold data that is related in some way
  • Control flow is the order in which a computer executes code in a script
  • When we create a function, we are defining a block of code that performs a specific task and giving it a unique name. However, just defining a function does not mean it will execute in our code automatically. To execute a function, we must call it in our code
